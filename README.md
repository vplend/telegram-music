#### Telegram Mini App для поиска и прослушивания музыки.

## Описание
TG Music - это веб-приложение на базе Next.js, работающее как Telegram Mini App. Приложение позволяет искать музыку и взаимодействовать с контентом прямо в Telegram.

## Основные возможности

- Удобный поиск песен
- Современный UI с Telegram UI Kit
